# spec (fork) v0.1.0

## Initial fork release

This is `github.com/woodie/spec`'s first tagged release: a fork of `sclevine/spec` (last tagged `v1.4.0`, December 2019), kept at the same import path (`github.com/sclevine/spec`) so it drops in via a `replace` directive with no call-site changes.

## What's added

- `it.Context()` -- passes through the real `t.Context()` for the running subtest.
- `it.T()` -- passes through the real `*testing.T`, for `Before`/`After` bodies that need `t.TempDir()`, `t.Skip()`, etc.
- `Describe`/`AsContext()` -- RSpec-style `describe`/`context` duality without a second, colliding type alias.
- `Var[T]` -- a generics-based typed box for sharing a value between a `Before` and its sibling `it` blocks.
- `Aliases`/`RunAliased` -- passes `describe`/`context`/`it`/`before`/`after` in as callback parameters, no per-file alias line needed.

See the README for the full rationale and a combined usage example, and `docs/COWORK.md` for the reasoning and upstreamable-PR framing behind each addition.

## Compatibility note

This release requires Go 1.24+ (`go.mod`'s `go` directive was bumped from upstream's `1.13` to cover `Var[T]`'s generics and `it.Context()`'s `t.Context()` dependency). Projects still on older Go should pin to upstream `sclevine/spec@v1.4.0` instead.
